Project Summary

The Spiel

WSU's CS Scholarship website needed an update, and CS students are alwasy eager to provide paid* labor.

* Paid refers to the students paying their tuition.

The Application

The Framework

.NET 6.0

This was a difficult upgrade, although our John managed to get the .NET upgrade done very quickly. Time was spent ironing out bugs and adding small components.

The Code

The Fun

  • Users log in to their account or create a new account. New accounts are of type student, but administrators can make them into judges or admins
  • Admins can edit and create award cycles, see and judge applications, edit or create award funds, and award scholarships.
  • Judges can see and judge applications
  • Students can apply for open award cycles, see and edit their applications.
